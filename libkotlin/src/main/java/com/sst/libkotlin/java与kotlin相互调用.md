代码转换
Java 代码转 Kotlin 代码
方式有2：
1）、直接将 Java 代码复制到 Kotlin 文件中，AndroidStudio 会出来提示框询问你是否转换
2）、打开要转换的 Java 文件，在导航栏点击 Code -> Convert Java File to Kotlin File
